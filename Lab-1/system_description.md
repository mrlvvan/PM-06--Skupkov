# Описание информационной системы

## 1. Назначение системы

Информационная система предназначена для демонстрации базовых механизмов работы с учетной записью пользователя:

- регистрация нового пользователя;
- вход в систему с использованием учетных данных;
- доступ к личному разделу после успешной авторизации;
- ввод и сохранение пользовательских данных в профиле.

Система реализована как учебный прототип для лабораторных работ.

## 2. Основные функции

1. Регистрация пользователя:
   - ввод email, пароля и имени;
   - проверка корректности email;
   - проверка минимальной длины пароля;
   - запрет регистрации с уже существующим email.
2. Авторизация:
   - вход по email и паролю;
   - обработка ошибок при неверных данных.
3. Сессия пользователя:
   - сохранение текущего пользователя;
   - восстановление состояния после перезагрузки страницы;
   - выход из системы.
4. Ввод данных:
   - редактирование имени пользователя в профиле;
   - сохранение изменений.

## 3. Предполагаемые пользователи

- студент (пользователь системы);
- преподаватель (проверка работоспособности и соответствия требованиям).

## 4. Используемые технологии

- Frontend: React (Create React App);
- Язык: JavaScript (ES6+);
- Стили: CSS;
- Хранение данных: localStorage браузера (учебный режим, без серверной части);
- Тестирование: Jest + React Testing Library (базовый smoke test).

## 5. Ограничения текущей реализации

- отсутствует серверная часть и API;
- пароли хранятся в localStorage (только для учебной демонстрации);
- отсутствует ролевая модель и расширенная политика безопасности.
